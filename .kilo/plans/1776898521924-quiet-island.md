# Plano: Completar Página de Vendedores com Edição e Detalhes

## Objetivo
Completar a funcionalidade da página de Vendedores criando páginas de formulário (novo/editar) e detalhes, adicionando rotas correspondentes e integrando com a lista existente.

## Análise Atual
- **Lista.vue**: Página de listagem de vendedores funcional com filtros e busca
- **API Backend**: Endpoints CRUD disponíveis (index, show, store, update, destroy)
- **Modelo**: Vendedor com campos nome_vendedor, contato_vendedor, usuario, status, etc.
- **Faltando**: Páginas de formulário e detalhes, rotas, navegação

## Plano de Implementação

### Fase 1: Criar Página de Formulário (Form.vue)
- Criar Vendedores/Form.vue baseado em Pedidos/Form.vue
- Campos: nome_vendedor, contato_vendedor, usuario, senha, status, email, telefone, comissao
- Modo criar/novo e editar existente
- Validações: campos obrigatórios, formato email, etc.
- Integração com API para salvar

### Fase 2: Criar Página de Detalhes (Detalhes.vue)
- Criar Vendedores/Detalhes.vue baseado em Pedidos/Detalhes.vue
- Exibir informações do vendedor (perfil, status, pedidos associados)
- Botão editar para redirecionar ao formulário
- Seção de pedidos do vendedor
- Ações: ativar/desativar status

### Fase 3: Adicionar Rotas no Router
- Adicionar rota /vendedores/novo → Form.vue (modo criar)
- Adicionar rota /vendedores/:id → Detalhes.vue
- Adicionar rota /vendedores/:id/editar → Form.vue (modo editar)

### Fase 4: Atualizar Lista.vue
- Conectar botão "Novo Vendedor" à rota /vendedores/novo
- Conectar "Ver Detalhes" no dropdown à rota /vendedores/:id
- Adicionar opção "Editar" no dropdown para /vendedores/:id/editar

### Fase 5: Testes e Validação
- Testar criação de novo vendedor
- Testar edição de vendedor existente
- Testar visualização de detalhes
- Verificar navegação entre páginas
- Testar permissões e validações

## Riscos
- Campos do formulário podem mudar conforme API
- Validações no backend podem diferir do frontend
- Relacionamento com pedidos pode precisar ajustes

## Critérios de Sucesso
- Página de listagem mantém funcionalidade
- Criação de vendedor funciona via formulário
- Edição de vendedor funciona via formulário
- Detalhes do vendedor exibem informações completas
- Navegação entre páginas funciona corretamente
- Build e testes passam sem erros

## Dependências Necessárias
- API backend já implementada
- Componentes UI já disponíveis (Button, Input, etc.)

## Estimativa de Tempo
- Fase 1: 2 horas (criar Form.vue)
- Fase 2: 2 horas (criar Detalhes.vue)
- Fase 3: 30 min (adicionar rotas)
- Fase 4: 1 hora (atualizar Lista.vue)
- Fase 5: 1 hora (testes)
