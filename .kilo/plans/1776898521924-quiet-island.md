# Plano: Implementar Impressão Completa do Documento Gerando PDF

## Objetivo
Implementar impressão do documento completo gerando PDF internamente e usando API do navegador para imprimir sem download, garantindo que todo o conteúdo seja incluído independentemente do tamanho da tela.

## Análise Atual
- **PedidoPrint.vue**: Componente com modal de preview usando window.print()
- **Detalhes.vue**: Usa select dropdown para imprimir, abre modal de preview
- **Impressão Atual**: Usa window.print() no HTML, pode não imprimir conteúdo além da tela visível
- **Problema**: window.print() limita impressão ao conteúdo visível na viewport
- **Objetivo**: Gerar PDF completo e imprimir via iframe para garantir todo conteúdo

## Plano de Implementação

### Fase 1: Reimplementar Geração de PDF para Impressão Completa
- Reverter para geração de PDF usando html2pdf.js para capturar todo o conteúdo
- Implementar impressão do PDF blob via iframe oculto sem download
- Garantir que todo o documento seja incluído na geração do PDF
- Manter modal de preview antes da impressão

### Fase 2: Manter Seletor no Header e Botão no Popup
- Manter select dropdown no header da página Detalhes para seleção de modo de impressão
- Garantir que o botão "Imprimir" no modal de preview esteja sempre visível e funcional
- Ajustar layout do modal para destacar o botão de impressão

### Fase 4: Otimizar Estilos para Impressão HTML
- Revisar estilos @media print para garantir que todo conteúdo seja impresso
- Remover estilos específicos para PDF (onclone callback)
- Testar impressão em diferentes navegadores e configurações

### Fase 5: Testes e Validação
- Testar impressão completa do documento HTML
- Verificar que preview modal funciona corretamente
- Garantir compatibilidade cross-browser
- Remover código não utilizado (PDF generation)

## Riscos
- Geração de PDF pode falhar em navegadores com limitações de canvas
- Impressão via iframe pode não funcionar em alguns ambientes
- Bibliotecas PDF podem aumentar tamanho do bundle
- onclone callback precisa cobrir todos os estilos para renderização correta

## Critérios de Sucesso
- PDF gerado inclui todo o conteúdo do documento
- Impressão funciona via iframe oculto sem download
- Select dropdown no header abre modal de preview
- Botão "Imprimir" no modal gera e imprime PDF completo
- Build e testes passam sem erros
- Compatibilidade com navegadores modernos

## Dependências Necessárias
- Reinstalar: html2canvas, html2pdf.js, jsPDF para geração completa de PDF
- Usar: iframe oculto para impressão sem download

### Fase 7: Ajustar Fluxo de Impressão
- Modificar Detalhes.vue para abrir preview modal ao selecionar opção de impressão
- Integrar modal de preview com botão de imprimir
- Remover geração automática de PDF, usar apenas via modal
