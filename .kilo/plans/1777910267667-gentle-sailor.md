# 📋 **Plano: Correção de Erros no Sistema de PDF**

## 🎯 **Objetivo**
Corrigir os erros que estão impedindo o funcionamento do sistema de geração de PDF:
1. **Erro 500**: TypeError ao tentar atribuir Collection a propriedade tipada como array
2. **PHP Deprecated Warning**: Aviso de depreciação no Logger do Laravel

## 🔍 **Análise dos Erros**

### **1. Erro Principal - TypeError na Linha 167**
```
Cannot assign Illuminate\Support\Collection to property App\Services\PrintData::$itens of type array
```

**Causa**: O método `preparePedidoData()` está tentando atribuir um `Illuminate\Support\Collection` (retornado por `DB::table()->get()`) diretamente à propriedade `$itens` que está tipada como `array`.

**Localização**: `api/app/Http/Controllers/PdfController.php:167`
```php
$data->itens = $itens; // $itens é Collection, $data->itens é array
```

### **2. PHP Deprecated Warning**
```
PHP Deprecated: Illuminate\Log\Logger::__construct(): Implicitly marking parameter $dispatcher as nullable is deprecated
```

**Causa**: O construtor do Logger não está marcando explicitamente o parâmetro `$dispatcher` como nullable, o que será obrigatório em versões futuras do PHP.

**Localização**: `/vendor/illuminate/log/Logger.php:43`

## 🛠️ **Soluções Propostas**

### **Fase 1: Correção do TypeError (Prioridade Alta)**
- **Arquivo**: `api/app/Http/Controllers/PdfController.php`
- **Ação**: Converter Collection para array antes da atribuição
- **Código**:
```php
// Antes
$data->itens = $itens;

// Depois
$data->itens = $itens->toArray();
```

### **Fase 2: Correção do Deprecated Warning (Prioridade Média)**
- **Arquivo**: `api/vendor/illuminate/log/Logger.php` (não editável diretamente)
- **Alternativas**:
  1. **Atualizar Laravel/Lumen** para versão compatível
  2. **Suprimir warnings** no arquivo de configuração
  3. **Configurar error_reporting** para ignorar warnings de depreciação

### **Fase 3: Validação e Testes**
- **Sintaxe**: Verificar que o código PHP não tem erros
- **Tipagem**: Garantir que as conversões de tipo estão corretas
- **Funcionalidade**: Testar geração de PDF após correções
