# Plano: Corrigir Erro de Template no PedidoPrint.vue

## Objetivo
Corrigir o erro "Element is missing end tag" no componente PedidoPrint.vue causado por seções duplicadas e tags de fechamento faltantes.

## Análise do Problema
- Erro: "Element is missing end tag" na linha 128 (div principal)
- Causa: Estrutura do template corrompida com seções duplicadas e divs não fechadas
- Problemas identificados:
  - Seção "Order Summary" (linhas 204-217) não fechada
  - Seção "Items" duplicada (linhas 253-283)
  - Estrutura geral desorganizada com comentários e divs fora de ordem

## Plano de Correção
### Fase 1: Remoção de Duplicatas
- Remover seção "Order Summary" duplicada (linhas 204-217)
- Remover seção "Items" duplicada (linhas 253-283)
- Limpar comentários e espaços desnecessários

### Fase 2: Correção de Estrutura
- Garantir fechamento adequado de todas as divs
- Manter ordem correta: Header → Client Info → Items → Order Summary → Additional Info → Signatures → Footer → Button
- Verificar nesting de elementos condicionais

### Fase 3: Validação
- Executar build para confirmar correção
- Verificar renderização do componente
- Testar geração de PDF

## Riscos
- Remoção de seções pode eliminar conteúdo importante
- Reordenação pode afetar layout e estilos
- Possível perda de funcionalidades

## Critérios de Sucesso
- Build executa sem erros de template
- Estrutura do template é válida e bem formada
- Componente mantém todas as funcionalidades originais
